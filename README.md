Aplikasi manajemen toko berbasis Java Swing dengan database MySQL.  

Dibuat untuk memenuhi tugas akhir mata kuliah Pemrograman Berorientasi Objek (PBO).
berikut adalah beberapa fitur yang saya buat:
- Registrasi user dengan upload foto profil
- Login user
- CRUD produk (create,read,update,delete)
- CRUD kategori produk
- Auto-generate ID untuk user, produk, dan kategori
- Tampilan modern dark mode

tools yang saya gunakan:
- Java Swing (GUI)
- MySQL (Database)
- JDBC (Koneksi database)
- IntelliJ IDEA
